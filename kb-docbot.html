KB DocBot

AI-assisted documentation updates from code changes.

What it does:

  • Reads recent GitHub commits and pull requests
  • Checks the documentation repo for affected pages
  • Drafts suggested Markdown updates
  • Opens a pull request for human review
  • Posts a Slack summary when review is ready

Why I built it:

Documentation drifts when product changes faster than help content. This tool reduces that gap without letting AI publish unchecked.

Human-in-the-loop by design:

AI can detect, draft, and explain. A person still owns accuracy, tone, and release timing.
